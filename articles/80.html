On the ☰ menu, there is a toggled command that switches between QR-code modes. These modes are used to respond when a stockd QR-code1 is scanned.

click to set modeQR: the user has never set a mode for QR-code reads.

edit inventory itemQR: when a stockd generated label is scanned, the user is directed to the inventory editing tool.

verify inventory itemQR: when a stockd generated label is scanned, the item is marked as verified today by the user who just scanned it.

NOTES
  1. In the inventory record, use +label to generate a printable QR-code label.
