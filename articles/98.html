Task search can quickly locate tasks using the text input and the filters.

options
+active: show all active1 tasks.
+new: start a new task.
+reset: reset the filters to default state.
text
searches for tasks with this string in the title or the attached notes
priority
filters on low/normal/high2
assigned
filters on the user assigned to the task
status
filters on the current state of the task3
NOTES
  1. A task is considered inactive when the status is “done” or “cancelled”.
  2. Set this to “all” with no other filters (click +reset first) to get a list of all tasks.
  3. The possible values are set by stockd.
