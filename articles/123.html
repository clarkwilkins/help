This tool has eight possible search modes that return matching cost sources.

name/item
If you want to located a cost source via an item reference, use this mode. It finds cost sources linked to an item that matches your search terms on name, part number/sku, serial number, or item number. Any source linked to a matching item is returned.
unassigned cost sources
This search returns any defined cost source that has not been linked to an inventory item.
source expenses (open/locked)
The two options for source expenses return the cost source and the linked item. The distinction here is made on whether the cost source is open (unlocked) for further changes.1
source jobs (open/locked)
The two options for source jobs return the cost source and the linked item. The distinction here is made on whether the cost source is open (unlocked) for further changes.1
source groups (open/locked)
The two options for source groups return the cost source and the linked item. The distinction here is made on whether the cost source is open (unlocked) for further changes.1

Results show the matching cost sources with details on the group and linked item(s).2

NOTES
  1. It's good policy to “lock” a source once the costs are settled. This should not be updated again in normal circumstances.
  2. Only source groups should have multiple item links. More information is here.
