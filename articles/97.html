Once a task is created, a few more fields and options are available. All tasks have the following components.

options
+search: switch to the task search engine.
+nudge: send a reminder about the task to the assigned user.
+new: start a new task.
title
This is the task name. (mandatory)
priority
low/normal/high
assigned
The user assigned to the task.
status
The current state of the task.1
followup
A date selector that can create a followup event for this task.

The task can also be deleted here.

NOTES
  1. Values are defined by stockd.
