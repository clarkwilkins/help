This analysis examines the movement of a set of items grouped by a common SKU or part name1. >Data is gathered from the item return records — specifically, the exchange item field.

In the first part, each link has the item as it left inventory, and. where applicable, the exchange item (or restock) returned. When reviewing this, look for items which have this pattern:

  • The item left inventory.
  • There is no exchange item.
  • You were expecting an exchange.

This may indicate an order that needs to be rebilled for failure to return the exchange.

The second part of the search concerns the lifecycle of each item. In the initial view, you can see:

  • The inventory item number
  • When the item was first stocked
  • When it left inventory
  • The number of recorded history events

Clicking on the item or dates takes you to the details record. Clicking on the events count shows (or hides) the related events list.

NOTES
  1. Part name searches may be less useful if the search term is broad and returns more than one SKU.
