The default view of tasks is the search engine, set to show everything that’s open. To run more advanced queries, use +search. Filters are inclusive – a task must match all criteria to be shown.

text
Searches for the text in any task title or notes.
priority
Filters by high, medium, or low priority.
assigned
Filters by the user assigned to the task.
status
Filters by the task status.

Results show the following data:

  • Date (assigned) and task name.
  • [aging in days]d since task was opened.
  • The task priority.
  • The user assigned to the task.
  • The task status.

    • Click on any task to bring up the task editor.

    NOTES
    1. Task status is defined by servicd.
