The account registers are normally accessed via the Chart of Accounts. Clicking on any account in the account name or balance field to get to the register.1

standard register mode

The register’s normal mode has four columns:

date
The transaction’s posting date.2
transaction
The transaction type.
amount
The amount of the transaction line which posts in this account.
balance
The running balance.

Each line links to the source transaction, so you can switch over easily.3

transaction locking mode

If you switch to this mode, you can add or remove transaction locks.4 If a given transaction has a lock from another user that is at (or above) your level, the entire row will be locked and you will not be able to change it.

The view has four columns:

id
The transaction’s ID.5
date
The transaction’s posting date.2
amount
The amount of the transaction line which posts in this account.
details
The top line is a link to the source item (charge, payment, etc.). The second line allows adding or removing a lock (if possible).

NOTES
  1. Clicking on the account number will send you over to the account editor instead.
  2. We differentiate between the date a transaction is created (i.e. a check’s printed date) and the date it posts in the statement/register. The latter is the posted date – that’s what we show here.
  3. The entire row is a link so you can “right-click” to open it in a new tab.
  4. Locking is also possible in the transaction analyzer.
  5. This cell links to the transaction analyzer for more advanced editing and analysis.
