Toolkits are sets of specific inventory items grouped together as a kit.

To start a new kit, set a name, condition, and add the first item from inventory. If the item number is valid, the kit can be established as this item.1 All other items in the kit will be set to follow it, and the entire set of items will constitute the kit.

Upon creation, or when a kit is selected, the kit editor is shown.

NOTES
  1. A kit cannot be started with an item that's already established as the leader for another kit.
  2. Other items in the kit are shown in the +items view.
  3. Available status values are set by stockd.
  4. The history indicates who did the audit (set this field). If it wasn't you, don't make this change.
