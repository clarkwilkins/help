This section is where you define the settings for a specific user context.1 The selection is made from choose settings area.The settings available in the configuration are as follows:

server
Each Simplexable customer is assigned a private server address.2
level
Access levels are 1-9. Higher level users gain access to more privileges and can lock out changes by lower level users.
can access this service
This determines if the user can access the service. When this is switched off, the user cannot reach this service within this company.
can manage customers
This determines if the user can access customer management within this service.
can manage users
This determines if the user can access customer management within this service.
email
This setting can use an alternate email address when the user is working in this particular configuration.3
notes
You can add optional notes about this group of settings.
NOTES
  1. Specific means this user, this company, this service.
  2. Be careful here, as putting in the wrong server will disable this user's access to the service.
  3. The default is the user's primary address.
