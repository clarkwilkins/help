Basic inventory search looks for matches on part name, part number, item number,1 and serial number. The search is inclusive — all potential matches are listed.

There are three main search filtering options and two special conditional options.

condition
The default option returns only items that are in stock. Other conditional filters are available to expand (“all items regardless of status”) or refine the search.

When using conditional filtering, items which are followers will only be included if the leader item qualifies for this conditional filter.2
location
Filters the results to a specific location.
include notes
The search is expanded to include items connected to notes where the search terms appear.
conditional options
+everything: shows all items in the search set found, regardless of item condition (default)
+good: only shows items in good condition

There are also some special search modes.

range based search
Enter x:y where x < y to get all items in a numerical range.
location
Searching on a location with no regular search terms results in a list of all items in that location.

You can click on any item to view inventory details or use +verify to do a quick item verification.

All search results include sku trend analysis.

NOTES
  1. Of these four parameters, only item number is not a wild-card search. It only returns exact matches.
  2. The reason is that follower items inherit the condition of the leader while they remain a follower, but revert to their original condition if the connection to the leader is removed.
