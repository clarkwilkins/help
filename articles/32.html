All users can set a preferred background color to be associate with their comments. To make a selection, go to “current”, then access your settings via ☰ on the top right corner.

Click on the color block above “preferred color” and make your choice from the palette. Your new color is saved immediately.

Within the discussion, colors are assigned as follows:

  1. If you have a color preference, you alway get the color reservation by default in your own view of the discussion.
  2. topical goes through the comments in order and checks if the author has a color preference.
  3. If a preference is found, and has not been reserved by a prior commenter, this user gets the color they prefer.
  4. After all preferences have been fulfilled (where there is no conflict in step 3), all users in the discussion who do not have an assignment get one now as the first available color in the palette.

If two users in a discussion have both reserved this color, the assignment works like this:

  • Each user sees their own comments as this color when they are viewing the discussion from their own account.
  • Any other user sees this color assigned to the first of the two above who makes a comment in the discussion. The second user's comments will receive a semi-random color assignment from the unreserved colors that remain.
