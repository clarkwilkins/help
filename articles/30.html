The editor allows basic updating of a journal entry.1 Metadata options are:

date
The journal entry's posting date.
category
Most journal entries will use the standard class, but you can mark an entry as income, expense, or deposit.2 When either optional category is used, it only affects profitability analysis.

Updating transaction lines

When updating debits and credits, the respective totals must balance, or the transaction will not be saved. You can add additional debits and/or credits using +line.3

NOTES
  1. For more advanced options, see the transaction analyzer in tools.
  2. Be careful to properly set categories when journaling costs or income, but also be careful to not set categories when journaling from one income account to another, or from one expense account to another.
  3. Top-right corner.
