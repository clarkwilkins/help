Discussion settings will depend on whether you own1 the topic. You access them via the ☰ icon on the top menu.

owner settings
setting
effect
title
Allows you to change the topic title.
open/closed
Allows you to lock the topic so no further comments can be added.2 The topic now shows up as “archived”.
restricted
When this field is set, only the topic owner can see the participants list (members of this discussion).
removing users from a topic

The topic owner is the only one who can remove other users from a topic. When triggered, this function also removes all of the user's comments in the topic per our ownership policy.

adding new users to a topic

The topic owner is the only person who can invite others to the topic. topical will look first for users you are already connected to and list any of them who are not on this topic. You can click on any name found and add them.

The inputs for a new user are:

name
the targeted contact's name
email
the targeted contact's email
SMS/mobile3
the targeted contact's phone

When an invite is sent, if the email/mobile pair is found in the user list for topical, the existing user is paired and linked to the discussion.4 Otherwise a new user record is created and paired.

participant settings

Participants are limited to putting discussion-specific settings in place that override their default (account-wide) settings.

The participant can customize alert settings for this topic, and can remove themselves from the participants.5


NOTES
  1. The owner is the person who started the topic.
  2. This is reversible — only by the topic owner.
  3. The mobile number is optional, but recommended because it acts as a real-time notice for updates.
  4. This is subject to filtering out inactive contacts and blocked contacts.
  5. With the option to leave comments in place or remove them.
