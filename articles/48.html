In the edit item mode, you can update the quoted item, replace it, or remove it.

search terms
Use this to trigger a search for a different item to replace this one on the quote.
quantity
Change this number as needed.
rate 1
The “exchange” price.
rate 2
The “outright” price.
position
Changes the sequence/order of items on the quote.

You can also delete the quoted item. Changes are saved as they happen.

+item sets you up to add a new item.

+setup switches you to the quote setup editor.

+clear resets the current search terms
