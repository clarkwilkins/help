stockd can generate followups automatically by signaling to the Simplexable global API. The basic mechanism is set in company setup, and, from that point, becomes fully automatic.

setting followups for new sales quotes
  1. Go to ☰ > other tools > setup (company preferences)
  2. Set new quotes set a followup in X days to a value of 1 or more.

The setting is saved automatically. From this point, every new quote will have a followup set automatically for (step 2) business days from today.
