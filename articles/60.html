From the inventory edit screen, you can assign an item to an existing or new cost source.

existing source expense
  1. Select “existing source expense” from cost source type.
  2. Select the source expense.1
existing source job
  1. Select “existing source job” from cost source type.
  2. Select the source group.3
existing source group
  1. Select “existing source job” from cost source type.
  2. Select the source job.2
new source expense
  1. Select “new source expense” from cost source type.
  2. Add the new source name4
  3. Add the new source value if known.5
new source job
  1. Select “new source job” from cost source type.
  2. Add the new source name4
  3. Add the new source value if known.5
new source group
  1. Select “new source group” from cost source type.
  2. Add the new source name.6
NOTES
  1. This list will only have unclaimed source expenses. If none are available, you need to create a new one.
  2. This list will only have unclaimed source jobs. If none are available, you need to create a new one.
  3. If no source groups exist, you need to create a new one.
  4. New source expenses and jobs are easier to manage if you mention the item number in the name.
  5. This can always be edited later in cost source management.
  6. This value should be unique in the set of source group names.
