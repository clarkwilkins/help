The task editor has the following fields and controls:

+tools
Link to the tools manager.
+nudge1
Send a nudge to the assigned user about this task.
+new
Start a new task.
title
All tasks require a title — the name of the task.
priority
High/normal/low.
assigned
The user assigned to the task.2
status
The current status of this task.3
notes
Optional additional information on the task.
followup
Optional date to be reminded to check on the task.
delete this task
Option to remove the task.
NOTES
  1. See nudges for more information.
  2. A task can only be re-assigned to a currently active user.
  3. Task status is defined by servicd.
