Trial balance is a standard accounting report listing the total transactons in all debits and accounts for the entity. Since, accountd is designed to accomodate multiple entities in a single system, you can run this report on each one as needed.

entity
When you are using more than one entity, this control selects the one used for the report.
start
Starting date for the report. This defaults to January 1 of the current year. If set later than the end date, it will revert.
end
Ending date for the report. This defaults to December 21 of the current year. If set earlier than the start date, it will revert.

The definitions of debit and credit differ here from what you see internally to a transaction.

debits: assets, expenses

credits: liabilities, equity, revenue

It's quite possible to see negative values here. The report is a net change in the account over the period. If you draw out more from “Cash” then you put in over the selected period, that negative number will be reported.

Since the internal transaction engine cannot post an unbalanced transaction, this report is considered of little use other than possibly looking for misapplication of funds.
