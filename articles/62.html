You can create a new cost source before assigning it. This is useful, for example, to define the cost for an item you don't actually have yet. Alternatively, you may find you need a source now, and simply want to assign it to an item after creating it.

source/cost
Select a source type here.
name
Enter a name for the source (required).1
cost
You can enter a cost if known, or leave this for a future update.
notes
Optional additional information.
options
+search: show the cost source search tool
NOTES
  1. A name that references the connected inventory item can be very helpful. — “Repair of item 1234”.
