Email setup in servicd allows you to send the associated information to any valid email address. The default settings for to/cc/bcc are determined by the client-side email setup as explained here. You can modify these as needed. If your view is not current showing "cc" or "bcc" options, use +cc and +bcc to bring them into view.1

subject is a required field which is usually preset by stockd, but you can change it at will.

text is an optional field where you can put your comments as the main body of the email message.

Depending on the source of the email, different attachments are included as follows:

source
attachments
service report
The service report is sent along with all attached documents that are marked public.2 An image is displayed in the PDF automatically. All other documents are shown as links to your file repository.
billing report
The report is attached as a PDF.

NOTES
  1. If the default email settings for cc or bcc are not set on the master record, they are hidden by default.
  2. Public is the default setting. This is set at upload, but can be changed using document editing.
