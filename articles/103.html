Upon creation, or when a kit is selected, the kit editor is shown.

options
+email: send an email with the toolkit as an attachment
+pdf: create a toolkit PDF
+new: create a new toolkit
+items: switches to the kit items view
name
all toolkits have to have a name.
kit
The master item for this kit.1
complete
yes/no field indicating the kit completeness
condition
selector indicating the kit's status2
audited
when the kit was last checked3
expected
when the kit is due to be at its next destination
description
optional notes on the kit
active
yes/no
NOTES
  1. Other items in the kit are shown in the +items view.
  2. Available status values are set by stockd.
  3. The history indicates who did the audit (set this field). If it wasn't you, don't make this change.
