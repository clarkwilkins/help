The sales order maps a quote over to an actual sale. On the main order view, these are the sections.

order XXXX details
This section contains metadata about the order. (See below.)
attached items
This section shows all items on the order now.1

+quote [...] is a link to the originating sales quote.
notes
Notes linked to this quote are shown here.2 You can click on a note to edit it.

+note brings up the interface to create a new note.
documents
This section shows all documents linked to the order.3
create a new order
This section shows all quotes from the last 30 days. You can click on a quote here to generate a new order from it.

Now, a breakdown of the order details section:

options
+new: start a new order
+email: send this order as an attachment
+pdf: create a PDF of the order
+preview: view the formatted order
customer
The customer account linked to this order.
name
The order name/description. This is set by default, but can be modified as desired.
order status
A selector with values defined by stockd that indicates the current state of this order.4
purchase order
Optional input of a customer PO.
billing status
A selector with values defined by stockd that indicates the current billing state of this order.4
followup
An optional date selector for setting a followup on the order. This defaults to tomorrow when a new order is created.
delete the order
This button will remove the order and all related data on demand.
NOTES
  1. The fundamental difference between a quote item and an order item is this: an item on a quote refers to a type of item, while an item on an order refers to a specific item linked and used on the order. The quote refers to quoting an item of SKU 1234..., but the order refers to actual item XYZ with SKU 1234...
  2. Notes in stockd can simultaneously link to many other items besides quotes.
  3. Documents in stockd can simultaneously link to many other items besides orders.
  4. These selectors are useful for memorializing order conditions. They are intended to denote status, and do not affect any logic.
