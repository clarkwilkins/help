You can add as many job tickets as desired. Each time, you have to use the first three fields.

reason
The reason for this ticket.
assigned user
Only currently active users are allowed here.
initial status
The current status of this ticket. This is usually level 1 or 2 to begin.
notes
An optional field for adding contextual notes that are not visible to the customer.
NOTES
  1. The reason can change from ticket to ticket as desired.
