setup
This is where metadata describing the quote is kept. It includes the customer account, quote state, and terms of sale. The setup is explained in detail here.

-delete will remove the quote completely.

-duplicate will bring up an interface to make a copy of the quote.

+item will bring up an interface to add an item to the quote.
notes
Notes linked to this quote are shown here.1 You can click on a note to edit it.

+note brings up the interface to create a new note.
preview
The items on the quote are shown here along with the current terms and conditions. This is where you can click on a quoted item to edit it.

+email will bring up an interface to send the quote to any email address.2 The structure of the quote email is discussed here.

+pdf will create the same PDF that's used in the email version, but in a format where you can save/use it by itself.
history
The change history for the quote.

NOTES
  1. Notes in stockd can simultaneously link to many other items besides quotes.
  2. Management of email addresses for clients is discussed here and here. Default settings are considered email policies, but can be overridden at will.
