Parts requests can be edited, and, where necessary, deleted.1

options
+search: switches you to request search mode.
+duplicate: allows a quick copy of the request. Useful for demand-swarm situations with multiple clients looking for the same thing.
+new: switches you to new parts request mode.
sku/part number
Changes the part number associated with this request.1
part name
Changes the name associated with this part number.1
status
Changes the request status.
notes
Changes the request notes.
NOTES
  1. Making a change on part number or part name change the part name for every request using this part number. This is done to make the request data consistent when a (hopefully) better part name is found. This data is not considered trusted enough to change actual inventory items. You have to go to an item in the inventory with the part number and change it there to affect inventory SKU:name pairs.
