The job report is a single view with all tickets on the job. The sections are:

  • Customer name, po, and address
  • All tickets in the order they were booked
  • Billable hours if any
  • Public-accessible documents, if any1

This is all combined into a single PDF which you can print and/or email. You can generate a job report from the edit ticket or billing views.
