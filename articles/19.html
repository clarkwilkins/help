There are only two requirements for setting up a topic: the title and the first comment. If you create the topic at this point, you are the only participant.

It's more common that you want to go ahead and define one or more others that will be “invited” into the discussion. To set up these pending “participants”, click on the top line that reads “Click here to add more...”

adding new contacts

If you are not yet connected to anyone in topical, you will automatically be shown the new contact section. Otherwise, you use +add someone to open it.

The fields for adding a new contact are:

name1
The name of the person you are inviting.
email1
The email address of the person you are inviting.
mobile/SMS1
The number of the person you are inviting.
message
An optional message to be included in the invitation message.

Once you save the contact, they show up on the pending list — meaning they will be added when you create the topic.

adding existing contacts

Once you have one or more existing contacts, they are shown by default and can be selected with a click.

removing pending contacts

Any pending contact can be removed by clicking on them in the list.

topic creation

When creating the topic, the following steps happen:

  1. The topic is created.
  2. Your first comment is added.
  3. Each participant is evaluated. If the contact is in topical, their account is created.
  4. Each participant gets an invite message to the topic.2

NOTES
  1. These three fields must be present to use the contact.
  2. The SMS version has a link to the topic. The email version includes information about you (the creator) and the topic name.
