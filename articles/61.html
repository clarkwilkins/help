As discussed here, there are three types of cost sources to manage. Begin by locating the source.

search by name/item
This locates sources by wild-card match in the source name or the correlated item name.1
lists
Alternatively, you can bring up a subset of the existing cost sources and check +details to find the correct one.

Once you have the correct source, enter the editor.

source type
You can change the source type if needed.
name
This is a mandatory field.2
cost
A single fixed cost for the source.3 If you are editing a source job or source group, there will be an additional control above labeled +cost. This allows recording multiple costs as discussed below.
linked item
This is a link to the actual inventory item.4

If you are managing a source job or group, use +cost to add a new cost to the source, or click on an existing cost to edit it.5

amount
Enter zero for now if this is not certain yet.
date
The cost date.
final
A yes/no selection.
notes
Optional additional information.
options
  • You can save a new cost with add this cost to the source.
  • You can save changes to an existing cost with update the cost.
  • You can delete an existing cost with delete the cost.

The total cost of the job or group is recalculated to reflect your changes.

NOTES
  1. A name may reference multiple sources. Use +details to help find the one you want.
  2. A name that references the connected inventory item can be very helpful. — “Repair of item 1234”.
  3. In the case of a source expense, there is no further breakdown of details. It's a 1:1 relationship between one cost and one item.
  4. Only shown on source expenses and source jobs.
  5. You can always come back and change this later if the record is not locked.
