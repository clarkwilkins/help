Each inventory item is a unique record in the system with an internal ID and the item number you normally reference it by. stockd is primarily intended to manage discrete items, but there is a separate system that handles manufacturing inputs (consumables).

edit mode options
+label: generates a printable QR-code label for the item. This can be used with the QR-tool to enable the user to scan an item label and quickly verify or update it.
+similar: shows all other items in inventory with the same SKU.1
+new: opens the tool to create a new item.

There are six primary characteristics of an inventory item.

item ID
A unique 32-character ID used internally to reference a specific item.
item number
A unique number used to label and reference a specific item.
part number/sku
All items are assigned a SKU. For non-OEM items, you can use something custom or "NA".
part name
This is explicitly linked to the part number. If an item with part number “A123” is named “Widget", this affects all items with part number “A123”.
location
All items are always assigned a current location.
condition
All items are always assigned a current condition as discussed here.

Additional characteristics of items include the following:

serial number
An optional value usually designated by the manufacturer.
ok to sell
yes/no field indicating the availability of the item.
leader
Used to indicate this item is following another one.
cost source information
Many items will have known costs sources as discussed here. Help on assigning cost sources is here.
category
Items are classified as standard items, assets, or customer items. The latter two are considered non-salable.
stocked
The date the item record was created.
verified
The date the item was last verified.

The internal use section is an optional way to aggregate a collection of N items onto an internal job (not an order). This feature is experimental and not fully developed at this time.

The connections section will show any orders, return records, or repair jobs this item has been linked to.

The documents section allows you to link new documents or photos to the item. Documents can be multiply linked to a number of other data types in stockd.
