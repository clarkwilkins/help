Many records have specific notes fields. There are also universal notes which are a special class that can link to an unlimited set of objects. For example, a note about a repair may want to reference the item being repaired and the repair job itself. A universal note is created once, but can be linked to many things.

A note can be created from the following tools:

  • cost sources
  • inventory items
  • repair jobs
  • return records
  • sale quotes
  • sales orders
  • tasks

All notes originate with the switch +note. Creating a note links it to the current object.

To edit or delete the note, click on it.

To multiply link a note:

  1. Open the note editor.
  2. Click the advanced notes options button.
  3. Click +link.
  4. Select a link type.
  5. Run a search and find the item you want to connect.
  6. Click on the item and it's immediately connected.

To unlink a note from an item:

  1. Open the note editor.
  2. Click the advanced notes options button.
  3. Click -remove link next to the item you want to unlink.
