A topic is created by a user with a title and a first comment.

The owner can add one or more participants at creation or later on via the discussion settings. A list of the participants in the topic is shown unless the owner sets the restricted flag.

The contents of a topic are considered the “thread” — one title and N >= 1 comments.

Comments are added by participants,1 and the topic is only visible to the participants. Comments are only allowed when the topic is open.2

When the topic is updated with a new comment, participants are updated per their account settings3 — with an override by their discussion-specific settings if defined.

The owner can close a topic or delete the topic at their sole discretion.

who's on the topic and “nudging” them

Normally, you can click on “N members” in the top right corner and see the members of the discussion. From here, click on any other member, and you can send them a nudge — a short message and a link to this topic. It's quick way to try to trigger participants to engage in the conversation.

If the owner restricts the topic in discussion settings, the member list is not shown.

images and other uploads

You can add any image or other file subject to an 8 MB size limit. The files are mixed into the discussion with positioning based on when they were uploaded. On successful upload, the discussion reloads.

If the file is an image of type JPEG, GIF, or PNG, topical will show it inline in the discussion. Otherwise, it's only shown as a downloadable link.

Also, if a caption is available from the upload, it's included below the image or above the file link if the upload is not a displayable image.

comment editing/removal

Only the author/owner of a comment can update it. Click on the text of any comment you own and the editor is shown. You can update the content or clear and save to delete it.


NOTES
  1. Users who have been invited to the topic.
  2. Set by the topic owner.
  3. The default setting is to notify you by email and text message.
