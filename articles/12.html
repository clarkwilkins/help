The general ledger is a standard accounting report. The only setup control is the entity.1

For each account found, a ledger is shown with every transaction. The details column will include a link to the transaction details2 and to the actual transaction itself.3

NOTES
  1. If your setup only has one entity (company) running in it, the selector will be skipped.
  2. The transaction number links to the transaction analyzer.
  3. The actual transaction is the underlying journal entry, credit charge, payment, or check.
