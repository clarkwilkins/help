Journal entries search is accessed via tools:journal entries. The parameters are:

number
Searches for a journal entry with the transaction number.1 This filter overrides the other two, but is ignored if the number is not an integer > 0.
account2
All accounts available from all entities are listed here. It filters results to include only journal entries with at least one register line involving the selected account.
amount2
Amounts <= 0 are ignored. This filters results to include only journal entries where at least one line matches the amount.

  2. The amount and account filters are both applied if they have valid values.
