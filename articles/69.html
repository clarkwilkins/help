Parts requests are used to record market requests. They track who's asking for what, and whether you have it for sale. As such, they provide an invaluable historical perspective on market conditions.

To create a parts request:

  1. Access the tool from the ☰ menu.
  2. Select the customer who's making the request.1
  3. Enter the part number and wait for the stock check to run.2
  4. Click on the part name if it came up in step 3, or add the part name if necessary.3
  5. Select a request status.
  6. Add notes if desired.
  7. Save the request.
NOTES
  1. Use +new client to create the customer first, if necessary, and then return here.
  2. This is a quick check to see if you have it in stock. If so, STOP and switch to quoting it. Sales quotes will automatically file parts requests for all items on the quote.
  3. If the part name is already in the standard parts names table, it will show up here whether you have ever had one in stock or not.
