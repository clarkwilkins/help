Parts requests are used to record market requests. They track who's asking for what, and whether you have it for sale. As such, they provide an invaluable historical perspective on market conditions.

To create a parts request:

  1. Access the tool from the ☰ menu.
  2. Select the customer who's making the request.1
  3. Enter the part number or name,2 and wait for the stock check to run.3
  4. Click on the part name if it came up in step 3, or add the part name if necessary.4
  5. Select a request status.
  6. Add notes if desired.
  7. Save the request.
Interfacing with healthica

If you have set up an account on healthica.com and set up your credentials as discussed here, additional options will be shown that can load your parts request onto healthica as well. When using this option, your request is immediately distributed on the healthica network, and you do not need to file a separate request there.5

NOTES
  1. Use +new client to create the customer first, if necessary, and then return here.
  2. This search is flexible. You can use a whole or partial SKU or part name. In either case, the search will include all potential matches.
  3. This is a quick check to see if you have it in stock. If so, STOP and switch to quoting it. Sales quotes will automatically file parts requests for all items on the quote.
  4. If the part name is already in the standard parts names table, it will show up here whether you have ever had one in stock or not.
  5. Your request will be found in ☰:my requests as normal.
  6. You can make changes on healthica under “my interests”.
  7. By design, any change to the part name overwrites this field. You want to change this after you are done with the part name.
