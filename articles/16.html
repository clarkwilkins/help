Pending expenses are set in the expense editor. Generally, this happens when the expense amount is set to zero, but it is possible to force the flag to true on demand.

The report will sort any pending expenses first by group, then date, and finally ID. From here, you have three options.

move to the expense
The main body of each line has basic details on the expense and acts as a clickable link to it.
remove the pending flag
+posting will mark the expense as “posting”.
delete the expense
-delete will remove the expense altogether.

If the expense is locked, the +posting and -delete will not be available.
