topical respects data ownership. The user owns all of their own comments and settings.1 The topic owner controls every other part of the discussions that take place here.

discussion
All metadata about the discussion can only be changed by the owner.
participants
Users can remove themselves from specific discussions. Only the owner can invite a user into a discussion.
connections
Users can block any other user from connecting with them.
personal data
Only the actual user can edit their own personal data.

Additionally, users can

  • deactivate at any time, suspending their account but leaving existing comments in place
  • delete their account including the removal of all topics they own and all comments they make on other topics

NOTES
  1. Your account setting are found on the main page. Navigate to current and click on ☰.
