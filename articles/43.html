The quote setup section has several fields that describe the quote as a whole (metadata).

customer
This is a selector that links the quote to the client.
created
The date the quote started.
followup
A date selector that can create a followup event for this quote.1
active/inactive
A click-triggered toggle between active and inactive states. Generally, a quote that will not be converting is set to inactive.
confidence
A five-step rating of the “odds of success” in converting this quote to an order. This defaults to “uncertain”, and use is completely optional.
load preset terms
You can set up standard sets of Terms and Conditions in other tools and then load them in terms just below here.2
terms
The Terms and Conditions text for this quote. You can edit it as needed.2
NOTES
  1. When a quote is created, a followup is automatically created for the next day.
  2. The default text is whatever was used on the last quote for this client.
