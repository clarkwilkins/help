Order search allows location of sales orders by a variety of search filters.

+recent
Show all orders created in the last seven days.
+new
Shortcut to creating a new order.
search terms
Run wild-card search against order notes and items1, and an exact match search against the order number or connected item number.
client
Filters the search by the customer account.2

Click on any quote in the results to get further details.

NOTES
  1. Connected items are wild-card searched against the part name and number (sku).
  2. Run this filter by itself to see all orders for a single client.
