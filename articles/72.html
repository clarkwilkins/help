Parts returns records are created in the order edit screen using the selector return item expected.1

The main view for this tool is open returns. The main table has all open repair jobs with links to the item details, and to the detailed return items record.

Use the combination of inventory information and the returns records to manage getting exchange items back on time.

NOTES
  1. This is the only scenario where a return record makes sense — an item is sold and something is being returned, either as an exchange, a restock, or a DOA.
