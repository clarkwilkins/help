To add a new inventory item, you need five fields.

item number
A unique number used to label and reference a specific item.1
part number/sku
All items are assigned a SKU. For non-OEM items, you can use something custom or "NA".
part name
This is explicitly linked to the part number. If an item with part number “A123” is named “Widget", this affects all items with part number “A123”.
location
All items are always assigned a current location.
condition
All items are always assigned a current condition as discussed here.

Additional characteristics of items you can add here are:

serial number
An optional value usually designated by the manufacturer.
ok to sell
yes/no field indicating the availability of the item.
leader
Used to indicate this item is following another one.
cost source information
Many items will have known costs sources as discussed here. Help on assigning cost sources is here.
NOTES
  1. stockd automatically assigns a next available ID using your last entry as a starting point. It will increment until it finds an open ID.
